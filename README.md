organiza-cli
=============

Herramienta CLI para organizar archivos en carpetas `Images`, `Documents`, `Archives`, `Apps`.

Uso:

```bash
organiza --dir /ruta/a/carpeta
# o
organiza -d /ruta/a/carpeta
```

Instalación local (después de pack):

```bash
dotnet tool install -g --add-source ./bin/Release organiza-cli
```
